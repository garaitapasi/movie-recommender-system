# Movie Recommender System

A movie recommendation web app built using Streamlit, Machine Learning, and TMDB API.

## Features

✅ Movie recommendations  
✅ Movie details  
✅ Cast information  
✅ Review sentiment analysis  
✅ Interactive UI
